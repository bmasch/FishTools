This tool is used to estimate the power of detecting a significant difference in log(smolt abundance) due to habitat actions in a paired design study. This tool may be used to determine the sample sizes (numbers of sites and years) necessary to deliver a statistically reliable estimate of the effect of habitat actions on log(smolt abundance). In the paired design, it is assumed that each of N sites contains a control watershed, where no habitat action is taken, and a treatment watershed, where habitat is deliberately altered.

The mathematical derivations are given in this paper (pdf):

PAIRED– a tool to calculate the power of a design for monitoring fish response to habitat actions

and can be reproduced with the following R code:

# Program to implement a power analysis of a paired experiment
# as in Liermann and Roni (2008)
#N is the number of sites
#n1 is the number of Before years
#n2 is the number of After years
#theta is the true effect size which is the true shift in the difference
# in treatment and control log(smolts) between the Before and After periods.
#
lr<-function(vartime=1,varsite=1,N=20,n1=2,n2=8,theta=.02,alpha=0.05){
sd<-vartime*(1/n1+1/n2)+varsite
sd<-sqrt(sd)
se<-sd/sqrt(N)
cv<-se/theta
delta=theta
q<-qt(p=1-alpha/2,df=N-1)
power<-1-pt(q,ncp=delta/se,df=N-1)+pt(-q,ncp=delta/se,df=N-1)
return(list(vartime=vartime,varsite=varsite,N=N,n1=n1,n2=n2,theta=theta,alpha=alpha,
se=se,cv=cv,power=power))
}
#outputs
#se -- standard error
#cv -- coefficient of variation
#power -- probability of rejecting the null hypothesis of no treatment effect

Choosing Input Parameter Values

Time Variance (vartime)

The site-specific (time) variance. Must be a number greater than zero.

Site Variance (varsite)

The between-site variance. Must be a number greater than zero.

# Sites (N)

The number of sites. Must be a positive integer.

# Before Years (n1)

The number of years of experiment prior to the application of treatment. Must be an integer greater than zero.

# After Years (n2)

The number of years of experiment after the application of treatment. Must be an integer greater than zero.

Treatment Effect (theta)

The treatment effect, which is the true shift in the difference in treatment and control log(smolts) between the Before and After periods. Must be a number.

Type I Error Probabability (alpha)

The probability of rejecting the null hypothesis of 'no treatment effect' when it is true, usually set at 0.05. Must be a number between 0 and 1.


Obtaining Results

To calculate results for a given set of input parameters, click on the Calculate button in the lower left corner. A row will be added to the data table to the right.

To clear the values from the table, click on the Clear button.


Interpreting Results

Output parameters are those displayed on the right side of the results table (unshaded cells).

Std Err

The standard error of the estimate of the treatment effect.

CoeffVar

The coefficient of variation of the treatment effect. This equals the standard error of the treatment effect estimate divided by the true treatment effect.

Power

The probability of rejecting the null hypothesis when it is not true. This varies as a function of the true treatment effect.


Exporting Data

To export data, click the Export button. The data currently in the results table will be downloaded to your computer in comma-separated value (CSV) format. You may have to set your browser to recognize the .csv filetype and associate it with a program such as Excel.
